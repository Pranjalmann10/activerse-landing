# Game Images Directory

Place your game images in this folder with the following names:

## Required Image Files:

1. **power-climb.jpg** - For POWER CLIMB game
2. **mega-grid.jpg** - For MEGA GRID game
3. **laser-escape.jpg** - For LASER ESCAPE game
4. **battle-arena.jpg** - For BATTLE ARENA game
5. **air-shoot.jpg** - For AIR SHOOT game
6. **basket-ball.jpg** - For BASKET BALL game

## Image Requirements:

- **Format**: JPG or PNG
- **Recommended Size**: 400x300 pixels or larger (will be automatically resized)
- **Aspect Ratio**: 4:3 or 16:9 works best

## How to Add Images:

1. Copy your game images from your computer
2. Rename them to match the names above (e.g., `power-climb.jpg`)
3. Paste them into this `public/games/` folder
4. The images will automatically appear on the website!

## Note:

If an image is missing, the card will show a colorful gradient background as a fallback.
